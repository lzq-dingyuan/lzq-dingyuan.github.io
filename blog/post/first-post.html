来一张美丽的风景照片吧。

这张图在 assets 目录下。当然，你也可以使用自己的图床。

写一段代码（支持所有主流语言的高亮显示）：

#include <stdio>

using namespace std

void main(){
    cout << "hello world!" << endl;
}

或者，写一段数学公式：

\begin{equation}\begin{matrix}x'=x\cos\theta-y\sin\theta\newline y'=x\sin\theta+y\sin\theta\end{matrix}\end{equation}

对样式不满意，可以在 assets/index.css 文件中自行定义。
